# Cozy Dreamers

Cozy Dreamers is a cozy home décor and creative lifestyle website designed to inspire warmth, comfort, and creativity.  
The website showcases aesthetic home décor collections, lifestyle inspiration, blog content, memberships plans, and cozy living services.

---
## Features

-  Modern and responsive homepage
-  Cozy home décor product showcase
-  Lifestyle blog section
-  Membership plans
-  Login and account pages
-  Responsive design for desktop and mobile
-  Few Interactive elements using JavaScript

---
## Technologies Used

This project was developed using:

- **HTML5**
- **CSS3**
- **JavaScript**
- **Visual Studio Code**

## How to Run the Project

1. Clone the repository

```bash
git clone https:///shreeti-lut/Demo-CozyDreamers.git
```

2. Open the project folder in **Visual Studio Code**

3. Run the website using:
- Live Server Extension in VS Code  
OR
- Open `index.html` directly in your browser
